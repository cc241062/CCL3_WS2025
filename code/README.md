## Description
**Oops! I Can Cook** is designed to help inexperienced home cooks learn to prepare healthy, home-cooked meals. The app provides a combination of guided lessons, recipe creation tools, and interactive challenges to help users gain confidence in the kitchen.

---

## Features
- User profiles: create, edit, and personalize your cooking journey.
- Monthly Quest Mode: learn new recipes and complete fun cooking challenges.
- Recipe management: add, edit, and search for recipes.
- Step-by-step instructions: detailed guidance to help beginners succeed.
- Motivation: gamified quests and rewards to keep users engaged.

---

## Users and Use Case
**Target Users:** Inexperienced home cooks who want to eat healthier and gain confidence in the kitchen.  

**Use Case:** Users can:
1. Create a personal profile.
2. Explore pre-made recipes in Quest Mode.
3. Add and organize their own recipes.
4. Track progress and participate in monthly challenges.

---

## Goals
- Teach beginners how to cook from scratch.
- Build confidence in the kitchen.
- Encourage home-cooked meals over takeout.
- Keep users motivated through interactive quests and recipe creation.
